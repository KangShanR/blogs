# appollo configuration center

> appollo 配置中心的使用

## java 项目中使用 appollo 客户端列表信息

>  在使用中，客户端会将远程配置数据拉到本地来，防止断网类异常情况时不能获取到配置数据。本地路径：
- Mac/Linux: /opt/data/{appId}/config-cache
- Windows: C:\opt\data\{appId}\config-cache
