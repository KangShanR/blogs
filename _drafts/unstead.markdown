# 没有加的东西

#### DNS

 DNS ：Domain Name System，域名系统
 域名的作用是将原本很无趣的 ip 地址给映射成为有意义的名字。

 #### 怎么查看所访问的网站的 ip 地址？

 通过万能的控制台命令 `ping` ，输入此命令后接上域名后，控制台会在后显示所 ping 的网站的 ip 地址。
