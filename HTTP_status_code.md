# HTTP状态码（HTTP status code)
> 关于HTTP协议使用过程中常遇到的状态码与其代表的意义

**常见的4个HTTP的状态码：**
- 200：请求成功
- 301：资源被永久地转移到其他URL
- 404：请求的资源不存在
- 500：内部服务器错误

**HTTP状态码分类**

|分类|分类描述|
|:------:|:------|
|1**|信息，服务器收到请求需要请求者继续操作|
|2**|成功，操作被成功并处理|
|3**|重定向,需要进一步操作以完成请求|
|4**|客户端错误,请求包含语法错误或完成请求|
|5**|服务器错误，服务器在请求过程中发生了错误|

----------
**Note:**常见的分类如上，在编码过程中Tomcat返回状态码时，就可以根据状态码大致判定下bug在哪里了。
