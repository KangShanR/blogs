### C#数据类型 ###


- C#中，数据共有三种类型
	- value types 值类型
	- reference types 引用类型
	- pointer types  指针类型

关于命名空间 `namespace`，msdn官网上有一句：`Namespaces implicitly have public access and this is not modifiable.`翻译：命名空间隐式默认为public公共访问权限，并且不能修改这个公共访问特性。也就是说命名空间访问不受限；
